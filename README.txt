Hello!

'pegs.l' makes up the entire Peg Solitaire LM program. 

- - - - - - -

HOW TO RUN

In CLISP...

1. (load "pegs.l") to load the program

2. Use (ga-demo-sep) to demo the program using separation of the pegs as a
fitness metric. 

Or...

3. Use (ga-demo-peg) to demo the program using the number of remaining pegs
as a fitness metric.
